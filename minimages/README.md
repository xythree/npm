### Description
> 可以批量压缩当前目录下的图片，支持png、jpg、gif格式

### Install
_ _ _

```js
    npm install -g minimages
```

### Usage
_ _ _
```js

    //先cmd命令跳转到要压缩的图片目录,然后
    
    minimages
    
    or
        
    minimages png
    
    or 
    
    minimages jpg
    
    or
    
    minimages gif
    
    or
    
    minimages png jpg
    
```
